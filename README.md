# Bühler Online Store — Vue.js Assignment

## Overview

This project is a single-page web application built as part of the Bühler frontend assignment.  
It allows customers to browse products, view product details, and manage a shopping cart.

The implementation focuses on clean architecture, maintainability, and a polished user experience, while staying fully aligned with the original problem statement and user journey.

---

## Tech Stack

- Vue 2
- Vuex (state management)
- Vue Router (routing)
- SCSS (styling)
- SVG assets for icons and branding

---

## Features

- Section-wise product listing grouped by category
- Product detail pages with deep linking (`/products/:id`)
- Centralized cart management with quantity support
- Checkout page with line totals and grand total
- Real-time date and time in header
- Consistent SVG-based icon system
- Responsive, clean enterprise-style UI

---

## Requirements

- Node.js v16.x or v18.x recommended
- npm v8+

(This project was developed and tested using Node.js v16+)
